# Soufiane Radouane - Personal Portfolio

This is a personal portfolio website built to showcase my skills and projects as a Software Engineer. The site features a modern design with complex, performant animations.

## About This Project

This project serves as a live demonstration of modern web development techniques. It's built with a focus on user experience, performance, and clean code. The interactive hero section and smooth page transitions are designed to create a memorable and engaging experience for visitors.

## Tech Stack

*   **Framework**: [Next.js](https://nextjs.org/) (App Router)
*   **Language**: [TypeScript](https://www.typescriptlang.org/)
*   **Styling**: [Tailwind CSS](https://tailwindcss.com/)
*   **Animation**: [Framer Motion](https://www.framer.com/motion/)
*   **Deployment**: Vercel

## Getting Started

To run this project locally, follow these steps:

1.  Clone the repository:
    ```bash
    git clone https://github.com/your-username/my-portfolio.git
    cd my-portfolio
    ```

2.  Install the dependencies:
    ```bash
    npm install
    ```

3.  Run the development server:
    ```bash
    npm run dev
    ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result. You can start editing the page by modifying `src/app/page.tsx`.
